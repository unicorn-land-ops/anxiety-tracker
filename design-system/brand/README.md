# Unicorn.Land Brand Assets

Production-ready brand identity files.

## Files

| File | Description | Usage |
|------|-------------|-------|
| `holographic-dot.svg` | Animated holographic dot (standalone mark) | Web, interactive contexts |
| `holographic-dot-static.svg` | Static holographic dot | Print, social, static exports |
| `favicon.svg` | Simplified dot for browser tabs | `<link rel="icon">` |
| `wordmark.svg` | UNICORN.LAND wordmark (light) | Light backgrounds |
| `wordmark-dark.svg` | UNICORN.LAND wordmark (dark) | Dark backgrounds |
| `badge-light.svg` | "Made in Unicorn.Land" badge | Light sub-product contexts |
| `badge-dark.svg` | "Made in Unicorn.Land" badge | Dark sub-product contexts |
| `social-avatar.html` | Social media avatar template | Screenshot at 400x400 |
| `social-cover.html` | Social media cover/banner template | Screenshot at 1500x500 |

## The Holographic Treatment

The dot and wordmark use a smooth flowing rainbow gradient (opal, blue, purple, pink, gold, green) that shifts position on a 6-second cycle. This is the brand's signature visual treatment.

## Exporting

Social templates are HTML files. Open in a browser and screenshot to export as PNG. The cover template uses the animated rainbow wordmark.
